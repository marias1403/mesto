# Проект: Место

### Описание и функциональность проекта

Сервис Mesto — это интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

### Используемые технологии

* html
* CSS
* БЭМ
* flexbox
* grid
* медиазапросы
* JavaScript
